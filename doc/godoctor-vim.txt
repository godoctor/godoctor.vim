==============================================================================

The Go Doctor provides source code refactoring for programs written in the Go
programming language (golang).

This Vim documentation file is a stub.  Complete documentation for the Go
Doctor, is available at the Go Doctor Web site

    https://gorefactor.org/

which also includes instructions for installing the full documentation for
the Go Doctor Vim plug-in.
